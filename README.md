# CivicConnect (Previously Jan Seva 24x7)

**CivicConnect** is a decentralized, open-source intermediary platform designed to bridge the digital divide in India's grievance redressal ecosystem. It facilitates the structured dispatch of citizen complaints to **CPGRAMS** Nodal Officers.

## 🏛️ Mission
To empower non-tech-savvy and rural populations by providing a simplified, "on-behalf" filing gateway, ensuring that every citizen's voice reaches the relevant government authority.

## 🚀 Key Features
* **Identity Matching:** Local string-matching of self-declared Government ID data.
* **Smart Routing:** Automated department selection based on a curated `directory.json`.
* **Representation Mode:** Allows educated youth to file grievances for others.
* **Public Verification:** Reference IDs for tracking dispatch fingerprints via JSON database.

## 🛠️ Technical Stack
* **Language:** PHP 8.x
* **Database:** Flat-file JSON (Privacy-focused)
* **Mail Engine:** PHPMailer (SMTP)
* **CSS:** Custom Mobile-Responsive Grid
