# Contributing to CivicConnect

Thank you for your interest in improving **CivicConnect**. As an unincorporated public service, we rely on the collective intelligence of the citizen community to keep this gateway secure, updated, and accessible.

## 🤝 How You Can Help
1. **Directory Updates:** Helping us keep `directory.json` updated with the latest Nodal Officer emails for Union and State departments.
2. **Security Audits:** Reviewing our PHP/JSON handling to ensure citizen data remains private and protected against injection.
3. **UI/UX for Rural Accessibility:** Improving the CSS to make the form even easier to use on low-end mobile devices and slow 2G/3G networks.
4. **Localization:** Translating the interface into regional Indian languages to reach the last mile.

## 🛠️ Development Workflow
1. **Fork the Repository:** Create your own copy of the project.
2. **Local Testing:** Set up a local environment (XAMPP/WAMP) and use `DEMO` mode in `config.php` to test your changes.
3. **Submit a Pull Request:** Explain your changes and how they benefit the public interest.

## 📜 Code of Conduct
* **Privacy First:** Never push code that logs sensitive user data to external servers.
* **Neutrality:** The code must remain a neutral "Digital Postman." Do not add features that curate or filter grievances based on content.
* **Respect:** This is a non-political, public utility project.
